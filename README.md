# Swiggy Food Order Analysis – Excel Project

## Project Overview

This project analyzes food ordering patterns from the Swiggy platform using Microsoft Excel. The goal of the project is to transform raw food order data into meaningful insights through data cleaning, pivot table analysis, and interactive dashboard visualization.

The dataset contains information such as restaurant names, dish names, food categories, pricing, ratings, and rating counts. Using Excel’s analytical features, the project explores customer preferences, restaurant performance, pricing distribution, and rating trends.

## Project Structure

Dataset – Contains the raw Swiggy dataset used for analysis.

Analysis – Includes pivot table analysis, summarized metrics, and interactive filtering using slicers to explore trends in the dataset.

Dashboard – Interactive Excel dashboard that visually presents key insights derived from the analysis.

## Objectives

• Analyze restaurant performance based on customer ratings and popularity
• Identify the most popular dishes and food categories
• Study price distribution across different food items
• Evaluate customer rating patterns and trends
• Present insights through an interactive and visual dashboard

## Tools Used

Microsoft Excel
Pivot Tables
Slicers for interactive filtering
Charts and Data Visualization

## Key Insights

• Certain food categories dominate customer orders across restaurants.
• A small number of restaurants receive the majority of customer ratings.
• Popular dishes typically fall within a moderate price range.
• Most ratings are concentrated within the mid-to-high rating range.

## Workflow

1. Data Collection
2. Data Cleaning
3. Data Analysis using Pivot Tables
4. Interactive Filtering with Slicers
5. Dashboard Visualization
6. Insight Generation
